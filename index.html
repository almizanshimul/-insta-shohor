Answer: Javascript is a scripting language. it runs inside of a browser. JavaScript works on both client-side and server-side. The interpreter inside the browser which will be always built-in reads the javascript codes. It is a single-threaded language. Where HTML creates the structure of the website, CSS designs. There JS User makes the website interactive and dynamic to make the website look more beautiful. We can generate HTML and CSS content also by the help of javascript. For example, with a simple calculator, we can calculate the temperature from Celsius to Fahrenheit using JavaScript.

There are two types of storage available for the client and server side: local storage and session storage. These two storages are very important for a website. These two storages enhance the user experience of a website. The difference between these two storages is described below:

local and session storage
local storage session storage
Local storage can be accesed from any window Session storage can be accesed from same window
Local storage can be deletd after a fixed time Session storage can be deleted when window is closed
To save an item in local storage we write (localStorage.setItem('Hi','how are you')) To save an item in session storage we write (sessionStorage.setItem('Hi','how are you'))
